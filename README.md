# Limepie

Limepie is a bitmap font for programming, all glyphs should be easily
discernible.

![example](img.png)

## Installation

```sh
~ $ git clone https://github.com/toniz4/limepie	
~ $ cd limepie
~ $ mkdir ~/.local/share/fonts # if this directory doesn't exist
~ $ cp *bdf ~/.local/share/fonts/
~ $ fc-cache
```
